Cloning, Building and Running Snack
===================================

Clone project
--------------
::

 git clone https://github.com/xjrga/snack.git
 cd snack

----

Build project
--------------
::
 
 ant -f build.xml create-thin-application

----

Run application
---------------
::

 cd distribution/snack-1515
 java -jar snack.jar
