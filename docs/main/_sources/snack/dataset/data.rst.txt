Data
====

Snack uses data obtained from:
  * The US Department of Agriculture, Agricultural Research Service, Nutrient Data Laboratory, SR28 Food Database
  * The National Academies of Sciences, Engineering, and Medicine, Dietary Reference Intake Nutrient Recommendations Report

Data Repository
---------------

This is where the food items in xml format are stored. 

https://github.com/xjrga/AllFoods
