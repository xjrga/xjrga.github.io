Installation
============

1. Download `potatosql-70-app.zip <../../releases/potatosql-70-app.zip>`_
2. Unzip potatosql-70-app.zip
3. Change directory to potatosql-70
4. Run the jar file
    * On linux: java -jar potatosql-70.jar
    * On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
