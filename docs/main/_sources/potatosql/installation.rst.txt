Installation
============

1. Download `potatosql-105-app.zip <../../releases/potatosql-105-app.zip>`_
2. Unzip potatosql-105-app.zip
3. Change directory to potatosql-105
4. Run the jar file
    * On linux: java -jar potatosql-105.jar
    * On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
