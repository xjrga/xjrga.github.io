Installation
============

1. Download `snack-1360-app.zip <../../releases/snack-1360-app.zip>`_
2. Unzip snack-1360-app.zip
3. Change directory to snack-1360
4. Run the jar file
    * On linux: java -jar snack-1360.jar
    * On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
