Installation
============

1. Download `snack-1505.zip <../../releases/snack-1505.zip>`_
2. Unzip snack-1505.zip
3. Change directory to snack-1505
4. Run the jar file
    * On linux: java -jar snack-1505.jar
    * On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
