Installation
============

1. Download `colorscheme-02-app.zip <../../releases/colorscheme-02-app.zip>`_
2. Unzip colorscheme-02-app.zip
3. Change directory to colorscheme-02
4. Run the jar file
    * On linux: java -jar colorscheme_02.jar
    * On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
