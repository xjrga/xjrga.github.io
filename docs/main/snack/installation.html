Installation¶

  1. Download snack-1510.zip

  2. Unzip snack-1510.zip

  3. Change directory to snack-1510

  4. Run the jar file

    • On linux: java -jar snack.jar

    • On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
