Description
-----------

Snack Nutrition Software is able export your diet and meal plans to a xml text file which another instance of Snack can read from to replicate the results. In this way, users can share diets, foods and nutrition information that have worked for them. These text files can also be modified using a simple text editor.
Process

 1. First application exports its diet and meal plans to a xml text file.
 2. Second application imports the diet and meal plans from the xml text file and replicates the results.
