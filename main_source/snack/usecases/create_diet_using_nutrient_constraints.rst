Create Diet Using Nutrient Quantity Constraints
===============================================

.. figure:: ../images/create_diet_using_nutrient_constraints.gif
   :alt: create diet using nutrient quantity constraints
   :align: center
   :target: ../../_images/create_diet_using_nutrient_constraints.gif
   
   This is how we create a mix using nutrient constraints.

Nutrient constraints can be used to create a food mix that meets specific dietary requirement, for example, a minimum amount of fiber in the diet. Nutrient constraints specify the amount of nutrient a food mix should have.
