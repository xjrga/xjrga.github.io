Installation
============

1. Download `snack-1320-app.zip <../../releases/snack-1320-app.zip>`_
2. Unzip snack-1320-app.zip
3. Change directory to snack-1320
4. Run the jar file
    * On linux: java -jar snack-1320.jar
    * On windows: double click the jar file

Java 11 or greater is required on your operating system for software to run
